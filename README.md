# Simple Interest Calculator

A simple, lightweight web application (or script) designed to calculate the interest earned or paid on a principal amount over a set period. This tool is perfect for quick financial snapshots and learning the basics of interest-based growth.

---

## 🧮 The Formula

This calculator uses the standard simple interest formula:

$$I = \frac{P \times R \times T}{100}$$

Where:
* **$I$**: Total Interest earned/paid.
* **$P$**: Principal Amount (the initial sum of money).
* **$R$**: Annual Interest Rate (in percentage).
* **$T$**: Time period (expressed in years).

The total accumulated value (**$A$**) is calculated as:
$$A = P + I$$
